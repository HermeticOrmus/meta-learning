# Learning Science: Evidence-Based Methods

> What research actually shows about effective learning

---

## The Core Problem

Most learning advice is either:
- **Folklore** - "this worked for me" without evidence
- **Marketing** - selling courses, not teaching effectively
- **Outdated** - based on debunked theories

This document summarizes what cognitive science actually demonstrates works.

---

## The Big Ideas

### 1. Active Retrieval Beats Passive Review

**The Research**: Testing yourself is more effective than re-reading or highlighting (Roediger & Karpicke, 2006).

**Why**: Retrieval strengthens memory pathways. Passive re-reading creates "illusion of competence."

**Application**:
- Quiz yourself before looking at notes
- Explain concepts without references
- Use flashcards for retrieval, not recognition
- Teach to test understanding

### 2. Spaced Practice Beats Massed Practice

**The Research**: Distributed practice over time produces better retention than cramming (Cepeda et al., 2006).

**The Forgetting Curve**:
- Day 1: Remember ~100%
- Day 2: Remember ~50% without review
- Day 7: Remember ~20% without review
- With spaced review: Retain 80%+

**Optimal Intervals**:
```
Initial learning → Review Day 1 → Day 3 → Day 7 → Day 14 → Day 30
```

**Application**:
- Schedule review sessions at increasing intervals
- Don't cram before "tests" — space it out
- Brief daily practice beats weekly marathon

### 3. Interleaving Beats Blocking

**The Research**: Mixing different topics during practice improves learning vs. practicing one topic at a time (Rohrer & Taylor, 2007).

**Why**: Interleaving forces discrimination between concepts and strengthens retrieval cues.

**Blocking** (less effective):
```
AAAA → BBBB → CCCC
```

**Interleaving** (more effective):
```
A → B → C → A → B → C → A...
```

**Application**:
- Mix problem types in practice sessions
- Don't master one thing completely before moving on
- Review older material alongside new

### 4. Desirable Difficulties Enhance Learning

**The Research**: Making learning harder in specific ways improves long-term retention (Bjork, 1994).

**Effective Difficulties**:
- Spacing (vs. massing)
- Interleaving (vs. blocking)
- Testing (vs. re-reading)
- Generating answers (vs. being told)
- Varying practice conditions

**Ineffective Difficulties**:
- Confusion without resolution
- Overwhelming complexity
- Poor instruction

**Key Insight**: If learning feels too easy, you're probably not learning much.

### 5. Sleep Consolidates Memory

**The Research**: Sleep is essential for memory consolidation (Walker & Stickgold, 2006).

**Why**: During sleep, the brain replays and strengthens neural pathways formed during learning.

**Application**:
- Don't sacrifice sleep for more study time
- Learn challenging material before sleep
- Naps can help consolidate specific skills

### 6. Elaboration Creates Durable Memories

**The Research**: Connecting new information to existing knowledge improves retention (Craik & Lockhart, 1972).

**Why**: More connections = more retrieval paths = better recall.

**Application**:
- Ask "how does this connect to what I already know?"
- Create analogies and examples
- Explain why, not just what
- Build mental models, not fact lists

## The Dual Process Model

### Focused Mode
- Concentrated attention
- Sequential, logical thinking
- Works on familiar problems
- Uses established neural pathways

### Diffuse Mode
- Relaxed attention
- Big-picture, pattern-matching
- Works on novel problems
- Creates new connections

**Both are necessary**. Switch between them:
```
Focus (25 min) → Break (5 min) → Focus → Break...
```

**Diffuse mode triggers**:
- Walking
- Showering
- Light exercise
- Sleep
- Mind wandering

## Deliberate Practice

### What It Is (Ericsson)

Deliberate practice is NOT just putting in hours. It requires:

1. **Specific Goal** - "Improve X skill" not "practice"
2. **Focused Attention** - Full concentration, no multitasking
3. **Immediate Feedback** - Know if you're doing it right
4. **Push Beyond Comfort** - Work at the edge of ability
5. **Repetition with Refinement** - Repeat while adjusting

### What It Isn't

- Playing through songs you already know
- Coding features you're comfortable with
- Reading without testing understanding
- Practicing bad habits consistently

### The Key Insight

10,000 hours of practice doesn't guarantee expertise.
10,000 hours of **deliberate practice** might.

**Quality >> Quantity**

## Cognitive Load Theory

### Types of Cognitive Load

**Intrinsic Load**: Inherent complexity of the material
- Can't reduce, must manage
- Break into smaller chunks

**Extraneous Load**: Unnecessary complexity from poor instruction
- Reduce this aggressively
- Clear explanations, good examples

**Germane Load**: Mental effort for learning
- This is the goal
- Processing that builds schemas

### Managing Cognitive Load

1. **Simplify** - Remove unnecessary elements
2. **Sequence** - Simple to complex
3. **Scaffold** - Support, then remove support
4. **Space** - Don't overload working memory

**Working Memory Limit**: ~4 items at once

## Growth Mindset

### Fixed vs Growth (Dweck)

**Fixed Mindset**:
- "I'm not good at math"
- Avoids challenges (might fail)
- Gives up when difficult
- Ignores useful feedback
- Threatened by others' success

**Growth Mindset**:
- "I haven't mastered math yet"
- Embraces challenges (opportunity)
- Persists through difficulty
- Learns from criticism
- Inspired by others' success

### The Impact

Growth mindset predicts:
- Greater achievement
- More resilience
- Better responses to setbacks
- Willingness to take on challenges

### Developing Growth Mindset

- Add "yet" to statements ("I can't do this... yet")
- Focus on process, not outcome
- Treat mistakes as information
- Praise effort, not talent

## Practical Synthesis

### Before Learning
1. **Prime**: Survey material, build framework
2. **Connect**: How does this relate to what I know?
3. **Set Goal**: What specifically will I learn?

### During Learning
1. **Engage Actively**: Take notes, explain, build
2. **Test Yourself**: Don't wait until "ready"
3. **Embrace Difficulty**: Hard = learning

### After Learning
1. **Review**: Spaced intervals
2. **Apply**: Use it in real context
3. **Teach**: Explain to others

### Environmental Factors
1. **Sleep**: Non-negotiable for consolidation
2. **Exercise**: Improves learning capacity
3. **Focus**: Eliminate distractions during practice

## Quick Reference

| Method | Effective | Ineffective |
|--------|-----------|-------------|
| Review | Self-testing | Re-reading |
| Practice | Spaced | Massed (cramming) |
| Topics | Interleaved | Blocked |
| Difficulty | Challenging | Too easy or overwhelming |
| Feedback | Immediate | Delayed or none |
| Mode | Focused + Diffuse | Only focused |
| Sleep | Full nights | Sacrificed for study |
