## Phase 4: ILLUMINATE (8 Lenses)

For each lens, create a dedicated page under "Master Learner Perspectives":

- [ ] **Da Vinci**: Cross-domain connections, visual hierarchy, observation journal
- [ ] **Tesla**: First-principles derivations, thought experiments, mental simulations
- [ ] **Franklin**: Decision tables, pros/cons analysis, practical checklists
- [ ] **Feynman**: Plain-English explanations, analogies, ELI5 versions, teaching script
- [ ] **Von Neumann**: Axiomatic foundation, theorem dependency graph, proof sketches
- [ ] **Aristotle**: Full taxonomy tree, essential properties, four causes
- [ ] **Musk**: Industry applications, engineering implementations, impact metrics
- [ ] **Oakley**: Chunking plan, spaced repetition schedule, practice progression

---

## Phase 5: APPLY (9 Methods)

For each method, create a dedicated page under "Learning Science Methods":

- [ ] **Encoding**: Relationship maps between concepts
- [ ] **Classification**: Hierarchical taxonomy
- [ ] **Pattern Recognition**: Recurring structures across theorems
- [ ] **Deliberate Practice**: Graded problem sets (easy/medium/hard/expert)
- [ ] **Focused-Diffuse**: Study session design
- [ ] **Metalearning Map**: Field overview (what/how/why/resources)
- [ ] **Observation**: Key phenomena and mental experiments
- [ ] **Tracking Discipline**: Progress tracking with milestones
- [ ] **Visualization**: Mental imagery for abstract concepts
